HTML

HTML or Hyper Text Markup Language is the standard markup language for creating webpages. It creates the layout of a webpage.

CSS

CSS or Cascading Style Sheets is style sheet language used to design a website. It describes how HTML elements are to be displayed on the website.

JavaScript

JavaScript is a language used to make a webpage dynamic and interactive. Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web.
